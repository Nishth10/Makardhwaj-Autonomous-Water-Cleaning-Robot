# 🌊 Makardhwaj: Autonomous Water Cleaning Robot (EcoSwachh)

## 🚀 Overview

Makardhwaj is an autonomous water body waste collection robot designed to remove both floating and semi-submerged waste from rivers and lakes.

This project was developed under the EcoSwachh initiative and later evolved into a startup supported by JNU incubation (AIC-JNUFI).

---

## 🏆 Achievements

* 🥇 Best Paper Award – International Symposium on Engineering (ISoE-2025)
* 🚀 Startup Incubation – AIC JNUFI
* 📄 Research-based implementation

---

## 🌍 Problem Statement

Water bodies are heavily polluted with plastic and non-biodegradable waste. Existing solutions are manual, inefficient, and unable to collect semi-submerged waste.

---

## 💡 Solution

Makardhwaj is an autonomous robotic system that:

* Collects floating and semi-submerged waste
* Uses ML-based object detection
* Works efficiently in rivers and lakes
* Provides real-time monitoring

---

## ⚙️ Technologies Used

* IoT Sensors
* Machine Learning
* Computer Vision
* Arduino & Raspberry Pi
* Pneumatic Bubble Lift System
* Conveyor Belt Mechanism

## 🧠 Innovation Highlights
